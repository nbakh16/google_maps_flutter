# track_user_movement

#### package used:
- https://pub.dev/packages/google_maps_flutter
- https://pub.dev/packages/location

#### features
- Current Location
- Draw polyline as character moves, every 10 seconds
- Update Marker location according to the user location
- Option to follow user
